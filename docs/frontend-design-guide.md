# GEO 智测工具 - 前端设计指南

## 1. 产品定位

**GEO 智能检测助手** - 面向黄金回收企业内容运营的智能分析工具

### 核心功能
- 评估内容对 AI 模型召回品牌的友好程度
- 识别内容与投放策略弱点
- 提供可执行的优化建议

## 3. 关键交互设计

### 3.1 品牌信息配置
- **首次使用**：必填项，引导用户完成配置
- **再次使用**：自动填充上次配置，可快速修改
- **保存配置**：支持保存多个品牌配置，快速切换

### 3.2 内容输入
- **文本粘贴**：支持富文本，自动统计字数
- **文件上传**：支持 .docx / .pdf / .txt / .md 格式
- **实时预览**：显示字数、段落数等基础信息

### 3.3 投放渠道选择
- **多选框**：支持选择多个渠道
- **智能提示**：根据选择的渠道，提示对应的强相关模型
- **渠道未定**：提供"渠道未定"选项，按基线50分评估

### 3.4 分析结果展示
- **分步加载**：显示分析进度（解析内容 → 评分计算 → 生成建议）
- **可视化图表**：使用进度条、雷达图展示各维度得分
- **可折叠区域**：详细诊断内容默认折叠，点击展开
- **高亮显示**：营销词在原文中高亮标注

## 5. 核心功能模块

### 5.1 品牌配置模块
```typescript
interface BrandConfig {
  brandName: string;        // 品牌名称
  brandPosition: string;    // 品牌定位
  serviceCity: string;      // 服务城市
}
```

### 5.2 内容分析模块
```typescript
interface ContentInput {
  text: string;             // 内容文本
  channels: string[];       // 投放渠道
  file?: File;              // 上传文件（可选）
}

interface AnalysisResult {
  overallScore: number;           // 综合评分
  friendlinessLevel: string;      // 召回友好度 (强/中/弱)
  dimensionScores: {              // 各维度评分
    authority: number;
    contentType: number;
    dataRichness: number;
    brandAnchor: number;
    length: number;
    marketingRisk: number;
  };
  modelScores: ModelScore[];      // 各模型评分
  marketingWords: MarketingWord[]; // 营销词命中
  suggestions: Suggestion[];      // 优化建议
  actionList: Action[];           // 立即行动清单
}
```

### 5.3 营销词检测模块
```typescript
interface MarketingWord {
  word: string;           // 词汇
  category: 'strong' | 'medium' | 'weak';  // 分类
  count: number;          // 出现次数
  positions: number[];    // 出现位置
  penalty: number;        // 扣分
}
```

## 7. 用户体验优化

### 7.1 引导流程
- **新手引导**：首次使用时显示功能介绍
- **示例内容**：提供示例文章，快速体验功能
- **帮助文档**：每个模块提供"?"图标，点击查看说明

### 7.2 性能优化
- **分析进度**：显示实时分析进度，避免用户等待焦虑
- **结果缓存**：相同内容不重复分析
- **懒加载**：详细诊断内容按需加载

### 7.3 错误处理
- **输入验证**：实时验证必填项
- **友好提示**：分析失败时给出明确的错误原因
- **重试机制**：网络错误时支持一键重试
